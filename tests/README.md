# AutoSocial AI Tests

This directory contains automated Java tests for the AutoSocial AI project using JUnit and Selenium WebDriver.

## Setup

1. Install requirements:
   - Java JDK 17 or later
   - Maven
   - Chrome browser

2. Make sure the main application is running on http://localhost:8000

3. Run tests:
```bash
mvn clean test
```

## Test Cases

1. Homepage Loading Test
2. Folder Monitoring Test
3. Content Generation Test
4. Post Scheduling Test
5. API Configuration Test

## Reports

Test reports will be generated in the `target/surefire-reports` directory
